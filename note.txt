Github Versioning Control Rules
// Newly use repository
- git init // menginisialisasi dari lcal git
- git add. // menambahkan semua folder/file baru kedalam local git
- git commit -m "your message" // untuk memberikan pesan terhadap folder/file yang kita tambah
- git remote add remote_name https//remote.link
- git push remote_name branch_name // untuk mengupload / mem-push folder/file yang sudah ditambahkan
